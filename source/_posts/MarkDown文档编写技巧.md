# 实现文章内部跳转

要跳转的目的地使用`span`标签进行包裹，跳转的出发地使用`[]()`进行包裹，方括号中填入的是跳转的文字内容，圆括号中填入的是span标签中id的值。

```html
[点击此处跳转跳转](#jump)

<span id = jump>跳转到这里<span>
```
